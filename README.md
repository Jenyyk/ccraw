# ccraw
ascii crows in the terminal

## Installation:
### Arch linux:
`ccraw` is available in the AUR:
```bash
paru -S ccraw
```
or if you suck:
```
yay -S ccraw
```

### from source:
`ccraw` is written in rust and can be built with cargo
```bash
git clone https://github.com/Jenyyk/ccraw
cd ccraw
cargo build --release
```
